# Aston Research - AI Consulting Website

Professional website for Aston Research, an AI consulting firm.

**Domain:** astonresearch.ai

## Overview

Single-page responsive website showcasing AI consulting services and company information.

## Structure

```
├── index.html          # Main website page
└── README.md          # This file
```

## Features

- **Responsive Design**: Mobile-friendly layout that works on all devices
- **Modern UI**: Clean, professional design with smooth animations
- **Sections**:
  - Hero: Eye-catching introduction with call-to-action
  - Services: Grid layout showcasing 6 core AI consulting services
  - About: Company overview and mission
  - Contact: Contact information and links

## Technology Stack

- Pure HTML5
- CSS3 (embedded styles)
- No dependencies or build process required
- Static site ready for deployment

## Local Development

Simply open `index.html` in a browser:

```bash
open index.html
```

Or use a local server:

```bash
python3 -m http.server 8000
# Visit http://localhost:8000
```

## Deployment

This static site can be deployed to:
- GitHub Pages
- Netlify
- Vercel
- Any static hosting service

## Color Scheme

- Primary: #4fc3f7 (Light Blue)
